## 打包
### rsrc
获取rsrc
```
go get github.com/akavel/rsrc 
```
生成*.syso
```
rsrc -manifest .\event.manifest -o event.syso
```
如果懒得弄，直接用项目提供的syso文件也可以，直接调用构建语句进行编译打包。

go build
